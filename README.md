# Task 3 - Marketing Funnel & Conversion Performance Analysis

## Internship: Future Interns | Data Science & Analytics

## Objective
Analyze marketing funnel data to identify conversion drop-offs, 
channel performance, and opportunities to improve lead-to-customer conversion.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts)

## Dataset
- 200 synthetic leads across 5 marketing channels
- Columns: Lead_ID, Date, Channel, Funnel_Stage, Converted, Revenue

## Key Findings
- Biggest drop-off: MQL → SQL stage (48% loss)
- Top performing channel: Paid Search (₹31,56,300 revenue, 23% conv. rate)
- Underperforming channel: Email (lowest conv. rate at 16%)
